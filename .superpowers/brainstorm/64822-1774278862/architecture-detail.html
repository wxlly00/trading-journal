Ce que fait chaque partie

EA MQL5 (dans MT5)
  • Détecte ouverture / fermeture de position
  • Envoie HTTP POST à Render avec clé API
  • Retry automatique si échec
  • ~50 lignes MQL5, 0 dépendance
FastAPI Backend
  • Reçoit les trades de l'EA (clé API secrète)
  • Calcule P&L, R:R, durée, drawdown
  • Expose API REST pour le dashboard
  • Auth JWT pour le dashboard web
Supabase (DB)
  • Table trades — chaque position
  • Table daily_stats — agrégats par jour
  • Table notes — annotations manuelles
  • Auth intégrée pour le login web
React Dashboard
  • Login → Dashboard → Journal
  • Graphiques Recharts (equity curve, etc.)
  • Filtres par paire, période, session
  • Mode responsive (mobile OK)
